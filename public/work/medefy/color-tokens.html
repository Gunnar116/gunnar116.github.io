Accessibility Color Token Reference

Medefy onboarding screen — WCAG 2.1 remediation. Green values in Recommended Hex are applied in medefy-onboarding.html.

Note on PRIMARY: The Figma table recommends #026BE6 (AA, 4.95:1 contrast with white). The implementation uses #004FBF (AAA, 7.22:1) to meet the stricter AAA threshold for interactive buttons. All other tokens match exactly.
